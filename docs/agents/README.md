# Логотипы агентов

32×32 пиксель-арт эмблемы для агентов системы. Каждый агент — иконографический
мотив в тёмном бейдже со скруглёнными углами (как app-иконка), с 1px контуром и
двух-трёхтоновой растушёвкой. Стиль серьёзнее скрамиков: без блоб-мордашек.

## Набор

| ID | Агент | Мотив | Бейдж |
|----|-------|-------|-------|
| `pm_agent` | PM-агент | Гэндальф-маг с палантиром (серая шляпа, белая борода, фиолетовый светящийся орб) | индиго |
| `meeting_summarizer` | Суммаризатор встреч | Документ-саммари с буллетами + эквалайзер «голосов» | бирюза |
| `board_audit_agent` | Аудит доски *(будущий)* | Kanban-доска под лупой с галочкой качества | янтарь |
| `shturm` | «Штурм» · проверка качества *(будущий)* | Боевой щит с золотой галочкой и молнией | багрянец |

Цвет бейджа кодирует роль: PM — управление (индиго), суммаризатор — спокойствие
(бирюза), аудит — внимание (янтарь), штурм — энергия/контроль (багрянец).

## Файлы

```
png/<id>.png        — ассет 32×32 (то, что идёт в продукт)
png/<id>@12x.png    — превью 384×384 (nearest, для глаз/презентаций)
png/contact.png     — общий контактный лист всех агентов
```

## Регенерация

```bash
python3 docs/agents/generate_logos.py
```

Зависимости: Pillow (есть в системном `python3`). Вся отрисовка — в
`generate_logos.py`: палитры и композиция мотивов лежат в функциях
`agent_pm` / `agent_meeting` / `agent_audit` / `agent_shturm`. Чтобы добавить
нового агента — пишешь функцию по образцу и регистрируешь в словаре `AGENTS`.
